OMAC Universal Data Analyzer

Professional Excel Add-in for Data Analysis & Compliance Dashboards

⚡ Install in Excel (Direct Method — Recommended)

1

Download manifest.xml from this repository

2

Open Excel Desktop → click Insert tab

3

Click Get Add-ins → click "Upload My Add-in" (bottom of dialog)

4

Browse to the downloaded manifest.xml → click Upload

5

An "OMAC Analyzer" button appears immediately in your Home ribbon — click it to open the panel

📊 Features

📊 Data Analyzer

Reads any sheet directly. X/Y pivot, multiple charts, slicers, sort, time formats.

✅ Compliance Module

Hand hygiene, audit compliance with speedometer, Met/Partial/Not Met criteria.

🎨 Interactive Charts

Multiple charts side by side, colour pickers, resize width/height per chart.

📥 Write to Excel

Paste the statistics table back as a new sheet in your workbook.

⬇️ Export HTML

Download a standalone shareable dashboard — works offline, no login.

🌙 Light & Dark theme

Toggle between light and dark mode with proper contrast.

🔧 Troubleshooting

!

Trust Center method not working? Use the direct Upload method above — it always works.

!

Add-in not loading? Make sure GitHub Pages is enabled (Settings → Pages → main branch → Save). Wait 2-3 minutes for it to deploy.

!

Charts not showing? Excel needs internet access to load Plotly. Try on a machine with internet.
